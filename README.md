# turtlebot3_wandou
 
使用方法

1. 下载代码

```
git clone https://github.com/wandouyu1/turtlebot3_wandou.git
```

2. 环境

```
ros noetic

```

3. 编译

```
    
```
cd turtlebot3_wandou
catkin_make
source devel/setup.bash
```

4. 运行

```
roslaunch turtlebot3_gazebo my.launch
rosrun control_wandou tiqu.py
